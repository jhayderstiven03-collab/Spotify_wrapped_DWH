
## Promt for Google Stitch
```
 Design a modern and minimal Spotify-inspired personal analytics dashboard web app with a premium aesthetic and clean user experience.

Style direction:

* Dark mode interface with elegant contrast
* Spotify green accent color (#1DB954)
* Minimalist UI with soft shadows, glassmorphism accents, rounded cards, subtle gradients, and smooth spacing
* Clean typography with strong visual hierarchy
* Modern SaaS dashboard style
* Spacious layout with balanced whitespace
* Responsive laptop-first design
* Avoid cluttered interfaces and excessive text
* Use smooth hover states and subtle microinteractions
* Professional, visually pleasing, and highly polished appearance

Pages to generate:

1. Login Page

* Centered layout
* Dark cinematic background with soft gradient or abstract music-inspired visuals
* Large app logo and title
* Single prominent “Connect with Spotify” button
* Minimal and elegant composition
* Smooth glowing Spotify-green CTA button

2. Profile Page

* Modern profile card layout
* Circular avatar with soft shadow/glow
* Display name as primary heading
* Email, country, followers count, and account type badge
* Premium/Free badge with distinct visual styling
* “Open in Spotify” external link button
* Sidebar navigation included
* Clean card-based sections with subtle animations

3. Dashboard Page

* Analytics-focused layout with modern dashboard composition
* Top 5 artists section:

  * artist image
  * artist name
  * animated popularity progress bar
* Top 5 tracks section:

  * track name
  * artist
  * formatted duration
* Peak listening hour widget:

  * large highlighted typography
  * visually prominent KPI card
* Dominant genres section:

  * horizontal bar chart
  * modern data visualization style
* Include elegant charts, cards, hover effects, and consistent spacing
* Dashboard should feel dynamic, premium, and data-driven

4. ETL Runner Page

* Technical monitoring page with clean developer-oriented UI
* DWH status table with modern badges and row styling
* Recent execution history table
* “Sync Now” button with loading/spinner state
* Real-time step-by-step execution log UI
* Terminal-inspired log styling but visually polished
* Use cards and sections to separate information clearly

Global UI Requirements:

* Consistent sidebar/navbar across authenticated pages
* Spotify-inspired visual identity without directly copying Spotify UI
* Use modern dashboard patterns similar to Linear, Vercel, Notion, or Stripe
* Maintain visual consistency across all pages
* Use realistic placeholder data
* High-quality UI/UX suitable for a professional portfolio project

Generate complete high-fidelity UI mockups for all pages.
```
